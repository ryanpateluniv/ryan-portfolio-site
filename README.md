## About

This site represents my work, interests, and technical direction as a computer science student focused on building intelligent systems from first principles.

It highlights:
- Research-oriented machine learning projects  
- Systems-focused engineering work  
- Academic achievements  
- Technical skill development  

The design philosophy is minimal, dark, and intentional — prioritizing clarity, typography, and structure over visual noise.

---

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Data-driven architecture
- Fully responsive layout

No heavy UI frameworks. No unnecessary dependencies.

---

## Architecture

The portfolio is structured around reusable UI primitives and modular page sections.

All content is data-driven via a central configuration file, allowing clean separation between presentation and information.

Designed for long-term extensibility and clean component integration.
